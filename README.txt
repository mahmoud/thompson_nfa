Regular Expression Matcher
Mahmoud Hashemi
CSCE428 Automata Project 1

==Files==
RegularExpressionMatcher.java - The file that does most of the work. Main method located here.
State.java - Class representing the states.
Transition.java - Class representing the transitions between states.

==Compilation==
To compile, simply run javac on RegularExpressionMatcher.java. Be sure to have the other two files in the same folder or path.

		javac RegularExpressionMatcher.java

		
==Usage==
To run the program, simply run java on the main class, RegularExpressionMatcher:

		java RegularExpressionMatcher
		
The program accepts standard input and outputs to System.out (console), per requirements.
